# Aegis-Vault: The Federal Reserve of Bitcoin

## 1. Vision
Aegis-Vault is a decentralized credit protocol on Stacks that transforms Bitcoin from a "passive store of value" into an "active, self-repaying asset." Users mint **aeUSD** against **sBTC** collateral, with native PoX yield automatically settling the debt over time.

## 2. Competitive Edge
- **0.0% Manual Repayments**: By leveraging native Stacks yield, the protocol handles the burden of debt settlement for the user.
- **Institutional-Grade UI**: A premium Next.js 16 dashboard with ultra-low latency and 3D glassmorphism.
- **Account Abstraction**: "One-Click" UX via sponsored transactions—removing the friction of gas fees for new users.
- **Clarity Secured**: Formal verification-ready contracts ensure that the protocol is mathematically solvent.

## 3. Product Roadmap (Executed)
- **Phase 1: Foundation**: SIP-010 `aeUSD`, `vault.clar` core, and a unified Clarinet environment.
- **Phase 2: Yield Engine**: `auto-repay.clar` logic and `oracle.clar` (Pyth/RedStone) integration.
- **Phase 3: Premium UX**: Next.js 16 (Canary) + Tailwind 4 frontend with Stacks.js integration.
- **Phase 4: Readiness**: 100% logic coverage, Subnet support, and security invariants.

## 4. Technical Stack
- **Smart Contracts**: Clarity 2.0 (Decidable, formal verification).
- **Frontend**: Next.js 16 (Canary), React 19, Tailwind CSS 4.
- **Infrastructure**: Stacks.js, Account Abstraction (Sponsored transactions).
- **Oracle**: Oracle-adapter for hybrid price feeds.
